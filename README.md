# Bio-Avatar Synthetic Dataset

This repository contains a synthetic dataset created for the purposes of the diploma thesis and the Bio-Avatar ecosystem.

The purpose of this dataset is to simulate biometric measurements that could be collected through wearable devices communicating with Wear OS intermediate layers and the Google Fit REST API. The dataset was created for testing, demonstration, and evaluation purposes, without using real personal or biometric data.

## Dataset Measurement Categories

The synthetic dataset includes the following biometric and wellness-related measurement categories:

- Heart Rate
- Sleep
- Steps
- Calories
- Heart Points
- Distance
- Activity Segments
- Active Minutes
- Height
- Weight

## Synthetic User Profile

For the creation of the dataset, the GPT-5.5 artificial intelligence model was used in order to generate a synthetic user profile. The generated profile represents a hypothetical user with the following characteristics:

- Male
- 35 years old
- No known medical history or chronic disease
- Exercises 3–4 times per week through different types of physical activity
- Works in an office-based environment

## Prompt Description

The dataset was generated through a prompt that instructed the model to create realistic biometric records for the above synthetic user profile. The prompt included constraints related to daily activity, sleep patterns, exercise frequency, timestamp structure, and consistency between the generated biometric values.

The goal was to create a realistic but fully synthetic dataset that could support the implementation and evaluation of the Bio-Avatar data flow, including data storage, daily summaries, role-based agent processing, and Mastodon-based output generation.

## Privacy Notice

This dataset is entirely synthetic. It does not contain real personal data, real biometric measurements, or information related to an identifiable individual. It is provided only for academic, research, testing, and demonstration purposes.
